# ResumeAI 开发计划 v2.0

## 项目概述

ResumeAI 是面向非IT行业求职者的简历优化工具，提供简历分析、优化建议、模板下载等功能。

## 技术架构

### 后端
- FastAPI + SQLite
- JWT认证
- AI分析: 通义千问API
- PDF生成: reportlab
- 文件解析: pdfplumber, python-docx

### 前端
- 纯HTML/CSS/JS（无框架）
- 简约黑白设计风格
- 移动端适配

### 部署
- 后端: Render (https://resume-ai-9tvi.onrender.com)
- 前端: Vercel (https://resume.bearai.bond)
- 邮件: Resend API
